⚠️ 免责声明

重要提示

本应用仅供参考，不能替代专业医疗诊断、治疗或建议。如有任何健康问题，请及时咨询专业医生。

1. 非医疗器械声明

本应用不是医疗器械，不提供医疗诊断、治疗或预防服务。所有内容仅供健康教育参考。

2. 紧急情况处理

如出现胸痛、呼吸困难、意识丧失、严重出血等紧急情况，请立即拨打120或前往最近的医院急诊科。

3. 内容来源

本应用内容基于公开医学资料和中医典籍整理，仅供参考学习。具体诊疗方案请以专业医生意见为准。

4. 用药提醒

任何药物调整、剂量变更都必须在医生指导下进行。请勿根据本应用内容自行调整用药方案。

5. 数据安全

您的健康数据仅存储在本地设备中，我们不会上传或收集您的个人健康信息。

最后更新：2025年1月 | 知体知医团队

🔒 隐私政策

1. 数据收集范围

我们收集以下信息用于提供个性化健康服务：

  • 基本信息：昵称、年龄段、性别
  • 健康信息：疾病史、过敏史、身体反馈记录
  • 使用数据：打卡记录、学习计划完成情况

2. 数据存储方式

✓ 所有数据仅存储在您的本地设备中

我们不会将您的健康数据上传到任何服务器，您的隐私完全由您自己掌控。

3. 数据用途

您的数据仅用于：

  • 生成个性化健康建议
  • 计算健康分数
  • 提醒您按时打卡和学习

4. 您的权利

您拥有以下权利：

  • 随时查看、修改您的个人数据
  • 导出您的健康档案
  • 清除所有本地数据（在健康档案页面操作）

5. 第三方服务

本应用使用Tailwind CSS CDN加载样式，该服务不会收集您的个人健康数据。
